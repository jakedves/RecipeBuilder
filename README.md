# RecipeBuilder (Ver 1.0 in progress)
## What is it?

ReciperBuilder is an iOS/iPadOS/MacOS application that allows users to create recipes easily and store them on their device for future reference.

## Technologies

- Swift 5
- SwiftUI 2
- Core Data

## Future Features

- Sync across iCloud
- Sharing recipes
- Public recipes
